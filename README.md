# AI NIDS Project

## AI-Based Network Intrusion Detection System

### Project Description
This project implements an AI-based Network Intrusion Detection System (NIDS) to detect malicious activities in network traffic using machine learning techniques.

### Objectives
- Detect network intrusions
- Improve network security
- Apply AI/ML concepts

### Technologies Used
- Python
- Machine Learning
- Scikit-learn
- Pandas
- NumPy

### How to Run
1. Install Python
2. Install required libraries
3. Run the file:
   python nids_main.py
